# Лабораторные работы по курсу "Операционные системы" МГТУ ИУ7. 6 семестр, весна 2025.

## Преподаватели

- Рязанова Н.Ю.

## Цель курса

Курс направлен на подробное изучение устройства работы Linux на уровне процессов, памяти, файловой системы и ядра. Курс включает изучение жизненного цикла процессов, механизмов их взаимодействия и алгоритмов планирования, а также на освоение архитектуры памяти, работы демонов и структуры файловой системы. В рамках курса также предполагается разработка модулей ядра, создание собственных обработчиков системных прерываний и изучение низкоуровневых механизмов ОС.
